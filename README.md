# YouTube Downloader

Простое приложение для скачивания видео и плейлистов с YouTube.

## Скриншоты

![Главное окно](screenshots/main.png)

![Очередь загрузок](screenshots/downloading.png)

![Настройки](screenshots/settings.png)

## Что умеет

- Скачивание одиночных видео и плейлистов.
- Параллельные загрузки с возможностью остановки.
- Темная тема, собственные диалоги и превью.
- Настройка папки загрузки и cookies; выбор качества (аудио, 360p, 480p, 720p, 1080p или максимум).

## Установка

```bash
git clone https://github.com/Laynholt/youtube_downloader.git
```

```bash
cd youtube_downloader
```

```bash
uv sync
```

```bash
uv run main.py
```
